Detailed Description

Variant stores value/type pairs in a convenient class.

The Variant class is both used to store and manipulate values internally. It can convert between different value types dynamically. It also provides a convenient mechanism to pass and retrieve values from the native Jinx API in a type-agnostic fashion.
