1/5/2023 A happy 2023 to everyone!  Jinx is up to version 1.3.10, with a series of minor updates and patches, such as an update to Catch2 for unit tests and documentation updates and fixes.  Jinx is stable and feature complete, so there’s not much activity recently, but that doesn’t mean it’s not being maintained. 4/1/2022 Jinx 1.3.5 has been released!  Jinx has added a native interface to call async functions, whch returns an ICoroutine-based interface, allowing a user to wait for function completion over time.  Additionally, better support for VS2022 has been added via CMake-based build scripts and .NET tools.
